# Bienvenido al Storybook de mi portafolio

Este es tu portafolio construido con React y configurado con Storybook. Aquí puedes ver y probar los componentes de tu aplicación de forma aislada.

## Contenido

- [Instalación](#instalación)
- [Uso de Storybook](#uso-de-storybook)
- [Componentes](#componentes)
- [Contribuciones](#contribuciones)

## Instalación

Para iniciar Storybook, asegúrate de tenerlo instalado en tu proyecto. Ejecuta el siguiente comando:

\`\`\`bash
npm run storybook
\`\`\`

Esto abrirá la interfaz de Storybook en `http://localhost:6006`.

## Uso de Storybook

Una vez que Storybook esté en funcionamiento, podrás ver todos tus componentes en el panel izquierdo. Haz clic en cualquier componente para ver su historia y prueba sus variaciones.

### Controles y Acciones

Algunos componentes tienen controles interactivos que te permiten modificar propiedades en tiempo real. También puedes ver acciones y eventos en el panel de "Actions".

## Contribuciones

Si deseas contribuir a este proyecto, por favor sigue los siguientes pasos:

1. Haz un fork del repositorio.
2. Crea tu rama (\`git checkout -b feature/NombreDeLaCaracterística\`).
3. Realiza tus cambios y realiza un commit (\`git commit -m 'Agrega una nueva característica'\`).
4. Sube tus cambios (\`git push origin feature/NombreDeLaCaracterística\`).
5. Abre una solicitud de extracción.

¡Gracias por contribuir!
